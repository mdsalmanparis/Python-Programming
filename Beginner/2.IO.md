# Python Input & Output Essentials

Clean, concise reference for Python `print()` and `input()`.

---

# Output with `print()`

## Basic Syntax

```python
print(value1, value2, ...)
```

```python
print("Hello")
print(42)
print(True)
print([1, 2, 3])
```

---

## Multiple Values

```python
print("Name:", name, "Age:", age)
```

- Automatically adds spaces between items.

---

## String Formatting

### f-Strings (Recommended)

```python
name = "Maya"
score = 95.5

print(f"{name} scored {score}%")
print(f"Next: {25 + 1}")
print(f"Pi: {3.14159:.2f}")
print(f"{score:>6.1f}")
```

### `.format()` Method

```python
print("{} is {}".format("Ravi", 30))
print("{1} before {0}".format("second", "first"))
```

### `%` Formatting (Old Style)

```python
print("%s is %d" % ("Ravi", 30))
```

---

## Controlling Output

### `sep` — Separator

```python
print("a", "b", "c", sep=" | ")
```

Output:

```text
a | b | c
```

Default:

```python
sep=" "
```

---

### `end` — Line Ending

```python
print("Hello", end=" ")
print("World")
```

Output:

```text
Hello World
```

Default:

```python
end="\n"
```

---

## Blank Line

```python
print()
```

---

## Special Characters

```python
print("Line1\nLine2")
print("Tab\tSpace")
print('She said "Hi"')
print("C:\\Users\\Name")

print("""Multi
Line""")
```

| Escape | Meaning |
|---|---|
| `\n` | New line |
| `\t` | Tab |
| `\\` | Backslash |

---

## Debug Shortcut (Python 3.8+)

```python
x = 10
y = 20

print(f"{x=}, {y=}, {x+y=}")
```

Output:

```text
x=10, y=20, x+y=30
```

---

## `print()` Defaults

| Parameter | Default | Purpose |
|---|---|---|
| `sep` | `" "` | Between items |
| `end` | `"\n"` | End of output |

### Example: Outputs
```python
name = "Salman"
age = 21
is_male = True
employment_details = {"Company":"Cognizant","Location":"Chennai","Exp":0.8}

#Printing
print("Hello!",name)
#Multiple Printing
print("Name:",name,"Age:",age)
#Print using f-String introduced in 3.6 for fast
print(f"Company Name: {employment_details['Company']}")

#Format Values: 4space & 2 decimal points
print(f"Experince: {employment_details['Exp']:>4.2f}")

#.format() Methods
print("Name is {} and the age is {}".format(name,age))
print("Name is %s and the age is %d" % (name,age))

#sep and end
print("Hello,", end=" ")
print("World!")

print("Apple","Orange","Banana","Kiwi",sep=' | ')
```

# Input with `input()`

## Basic Syntax

```python
name = input("Enter your name: ")
print(f"Hi {name}")
```

> `input()` always returns a string (`str`).

---

## Type Conversion

```python
age = int(input("Age: "))
height = float(input("Height: "))
```

### Important

```python
int("25")      # Works
int("25.5")    # Error
```

Use `float()` first if decimal values are possible.

---

## Safe Conversion

```python
try:
    age = int(input("Age: "))
except ValueError:
    print("Please enter a whole number")
```

---

## Multiple Inputs

### Split by Space

```python
x, y = input("Enter 2 numbers: ").split()

x = int(x)
y = int(y)
```

### One-Liner

```python
x, y = map(int, input().split())
```

---

### Split by Custom Delimiter

```python
data = input("Enter CSV: ").split(",")
```

Input:

```text
a,b,c
```

Output:

```python
['a', 'b', 'c']
```

---

## Default Value for Empty Input

```python
name = input("Name [Guest]: ") or "Guest"
```

If user presses Enter:

```python
name == "Guest"
```

---

## Common Behaviors

| Behavior | Example | Note |
|---|---|---|
| Always returns `str` | `input()` → `"5"` | Convert manually |
| Remove spaces | `input().strip()` | Cleans whitespace |
| `EOFError` | Ctrl+D / Ctrl+Z | End of input |
| `KeyboardInterrupt` | Ctrl+C | User cancelled |

---

## Common Gotchas

```python
# Risky
int(input("Num: "))

# Better
value = input("Num: ").strip()
```

```python
# input() automatically removes trailing newline
```

## Quick Validation Helpers

```python
s = input("Test: ")

s.isdigit()   # "123"
s.isalpha()   # "abc"
s.isalnum()   # "abc123"
```

# Quick Summary

| Function | Purpose |
|---|---|
| `print()` | Display output |
| `input()` | Read user input |
| `f""` | Modern string formatting |
| `.split()` | Split input into parts |
| `.strip()` | Remove extra spaces |
| `map()` | Convert multiple values |

### Example: Inputs
```python
name = input("Enter your name: ")
if name.isalpha():
    print(f"Hello, {name}")
    age_str = input("Enter your age: ")
    if age_str.isdigit():
        age = int(age_str)
        print("You're",(2026-age),"Born.")
    else:
        print("Age must be a number!")
else:
    print("Name should only letters.")
```
